# Web Calculator

A modern, accessible web calculator built with vanilla HTML, CSS, and JavaScript.

## Features
- Keyboard-first input support
- Light / Dark theme toggle
- Calculation history with reuse
- Safe math evaluation (no `eval`)
- Responsive layout for desktop
- Accessible UI with ARIA labels and focus states

## Tech Stack
- HTML5
- CSS3 (CSS Variables, Grid)
- Vanilla JavaScript

## How to Run
1. Clone the repository
2. Open `index.html` in your browser
